Open Source Projects

WhichCourse

A search interface to aggregate and display course reviews scraped from Carnegie Mellon's official faculty evaluation site in a manner that allows for easy side-by-side course comparisons.

Tools: Python, Flask, Scrapy, Selenium, Heroku
Links: Live (Heroku's dynos may be slow), GitHub Code & Docs, Blog Post

Tower Defense

A real-time strategy based game that pits players against waves of enemies that become increasingly difficult to stop. Gameplay includes 10 enemy waves and 4 towers with varying characteristics.

Tools: Python, Tkinter (standard GUI package)
Links: GitHub Code & Docs

Error Analysis

An extension of my summer research at UCLA's NanoCAD Lab that explored the robustness of various binary number representations with regard to soft errors from radiation and bitwise errors.

Tools: Python, Monte Carlo Simulations
Links: Research Poster, Research Paper
Awards: Semifinalist in the Siemens Competition, Presenter at UCSD Conference
